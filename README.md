# Detecção de Frutas usando IA (Roboflow + Python)

Este projeto implementa um sistema de **visão computacional** capaz de detectar e identificar três tipos de frutas:

- **Banana**
- **Maçã**
- **Laranja**

A solução utiliza o **Roboflow** para criação e treinamento do modelo e o **Python + OpenCV** para realizar inferência e exibir as detecções.

---

## Tecnologias Utilizadas

- **Python 3.8+**
- **Roboflow (API e plataforma de treinamento)**
- **YOLO (modelo de detecção utilizado no Roboflow)**
- **OpenCV**
- **Requests**
